What is 'pylacewing'?
---------------------
pylacewing is a set of modules for use with the Twisted framework so
you can communicate with the Lacewing relay protocol.
It works very well with games because of the lightweight protocol.

How do I install pylacewing?
----------------------------
Do the usual "python setup.py install" :-)
